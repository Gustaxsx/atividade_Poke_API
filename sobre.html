Início Buscar Pokémon Sobre

Sobre este Projeto

Este site foi desenvolvido como uma atividade prática para demonstrar o consumo de APIs (Interfaces de Programação de Aplicações) e a criação de um site web com múltiplas páginas.

Tecnologias Utilizadas

  • HTML5: Para a estrutura semântica do site.
  • CSS3: Para a estilização e o design responsivo.
  • JavaScript (ES6+): Para a interatividade e consumo da API.

API Utilizada

Este projeto consome dados da PokéAPI, uma API pública e gratuita que fornece informações detalhadas sobre o universo Pokémon.
